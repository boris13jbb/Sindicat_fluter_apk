# Diseño premium pantalla por pantalla — Sistema Integrado Sindicato

## Sistema visual base
- **Estilo:** premium, limpio, corporativo, moderno y orientado a dashboard móvil.
- **Paleta:** morado principal `#6847BE`, morado oscuro `#2B2265`, lavanda `#EEE5FF`, fondo claro `#FCF9FF`.
- **Tarjetas:** fondo blanco, bordes redondeados grandes, sombra suave y separación amplia.
- **Iconografía:** minimalista, consistente, en morado con fondos lavanda.
- **Navegación:** encabezados con gradiente, botones circulares de acción, barra inferior simple para módulos principales.
- **Jerarquía:** título grande, subtítulo descriptivo, métricas arriba, acciones en tarjetas y listas limpias.

## Reglas para implementación en Flutter
1. Crear un `AppDesignTokens` con colores, radios, sombras, padding y tamaños.
2. Crear widgets reutilizables: `PremiumHeader`, `PremiumCard`, `MetricCard`, `ActionTile`, `StatusPill`, `SearchField`, `PrimaryButton`.
3. Mantener `SafeArea` en todas las pantallas.
4. Usar `SingleChildScrollView` en pantallas con formularios o listas largas.
5. Evitar textos pegados a bordes: padding mínimo horizontal de 24 px.
6. Para Android, revisar barra de navegación inferior y usar padding adaptable.
7. Mantener consistencia de iconos por módulo:
   - Voto: urna / check.
   - Asistencia: persona con check / QR.
   - Socios: grupo de personas.
   - Auditoría: reloj / historial.
8. Cada pantalla debe conservar permisos actuales de navegación y reglas de seguridad existentes.
