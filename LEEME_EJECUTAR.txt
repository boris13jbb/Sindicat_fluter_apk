Para ejecutar la app sin el error de Git en Windows:

  1. Doble clic en:  run_chrome.bat
     (o en la terminal:  flutter run -d chrome)

  2. O en la terminal, desde esta carpeta (fluter_apk):
     flutter run
     y elige [2] Chrome (ya no aparecera Windows hasta que lo habilites).

La plataforma Windows (escritorio) esta deshabilitada porque el build requiere
Git en el PATH del sistema. Si mas adelante quieres volver a usar Windows:

  - Renombra la carpeta  windows.disabled  a  windows
  - Anade Git al PATH del sistema (ver BUILD_WINDOWS.md)
  - Cierra y abre la terminal/IDE y ejecuta:  flutter run -d windows
